# cis376-salon-app
CIS 376 Project 1: Salon Booking App
Author: Jasmine Morgan using VSCode IDE and further sources linked <br>
This project is a Github Pages site where you may log in/out, book an appointment, and view portfolios for a salon. <br> <br>
Github Repository - https://github.com/galaxydotco/cis376-salon-app <br>
Github Pages Link - https://galaxydotco.github.io/cis376-salon-app/ <br> <br>
My mother owns a salon, and I would like to take this experience to create a funcitoning or semi functional site for her to use, as she has no real website.
I have taken inspiration from a previous site I had built with zero functionality for a project proposal last year in my Github profile ciSalon repository. <br> 
- This site had been test on desktop using Chrome, Edge, Firefox, OperaGX, and Brave browsers.
- This site has been tested for mobile using Safari browser. 
- This site has been tested for use of "dark mode" browser extensions. 
<br> <br>

In attempt to gain bonus points, the search will make everything not applicable to the search disappear on the page. If you search "haircut", then sections within haircut will appear and any other options will be hidden!
